用户使用协议
本工具为免费、开源、面向技术研究的系统工具，仅限合法设备、个人学习与技术调试使用。

1. 用户必须拥有设备的完整所有权与合法使用权，不得用于非法破解、恶意修改他人设备、破坏系统安全、窃取数据等违法违规行为。
2. 本工具涉及 Root、分区读写、内核解包、重启、Bootloader 操作、ARB 检测等高风险操作，可能导致设备变砖、数据丢失、失去官方保修、系统异常。
3. 所有操作均由用户主动触发，风险完全由用户自行承担，开发者不对任何设备损坏、数据丢失、法律纠纷承担责任。
4. 禁止用于商业用途、二次分发、反向工程、去除版权信息、倒卖工具等侵权行为。
5. 用户需具备基本的 Android、Shell、ADB、Magisk、分区结构等知识，因操作不当造成的一切后果由用户自行负责。
6. 本工具仅从 GitHub 下载必要文件，无后台上传数据、无隐私窃取、无恶意行为。
隐私服务协议
本工具严格保护用户隐私，遵循最小必要、本地优先原则：

1. 本工具所有功能均在本地执行，不会上传任何设备信息、个人数据、文件内容、操作记录至远程服务器。
2. 工具仅访问系统分区、boot 镜像、xbl_config 等与功能相关的路径，不读取通讯录、短信、相册、位置、账号等隐私数据。
3. 工具仅在用户主动触发时从 GitHub 下载工具文件，无额外网络行为。
4. 工具不收集、不存储、不共享任何可识别用户个人的信息。
5. 卸载/删除工具后，本地产生的临时文件由用户自行清理。
用户须知
本功能涉及系统深度操作，可能影响设备安全与稳定性。
